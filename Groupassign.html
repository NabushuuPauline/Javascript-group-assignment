Capitalization

This transforms a text input or string to “UPPERCASE”, the reverse is also true in the case of “LOWERCASE" letters.
Sometimes capitalization is only required for the first letter of a given word, here we assign numbers to each letter and call out the letter we want.
We use;
.toUpperCase
.toLowerCase
.slice Or .split(when intending to use both)
Given;
Var string = "pauline enjoys discovering new music"

Click the button to convert the string to uppercase letters.

Click the button to convert the string back to lowercase letters.
